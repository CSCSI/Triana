Description of WriteGIF

WriteGIF takes in a TrianaPixelMap  (Triana's internal image type) and converts it into a java Image. This is converted into the GIF file format and then output to the specified file.

Using WRITEGIF

Double-click on the unit while holding down the Control key to pop up a file browser which allows you to choose the name of the file name for the GIF file.
