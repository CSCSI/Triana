GetGreen

Author : Melanie Rhianna Lewis
Input Type : TrianaPixelMap
Output Type : TrianaPixelMap
Date :
This UNIT generates a TrianaPixelMap containing just the green component of a source TrianaPixelMap. This allows you to carry out operations on just the green colour channel. You can combine the results with your original image using SetGreen
