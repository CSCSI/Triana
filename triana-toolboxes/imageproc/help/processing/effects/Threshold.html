Threshold

Author : Melanie Rhianna Lewis
Input Type : TrianaPixelMap
Output Type : TrianaPixelMap
Date :
Thresholds a TrianaPixelMap. Ie all pixels with intensities below a certain value are set to black and all other pixels are set to white. The threshold value is set using the Unit window.
