EnhContrast

Author : Melanie Rhianna Lewis
Input Type : TrianaPixelMap
Output Type : TrianaPixelMap
Date :
This Unit enhances the contrast of a TrianaPixelMap by expanding the range of the intensities of the pixels so that they lie between black and white. Ie it will expand the dynamic range of an image.
