GradientEdge

Author : Melanie Rhianna Lewis
Input Type : TrianaPixelMap
Output Type : TrianaPixelMap
Date :
This Unit is a Gradient Edge Detector. It looks for the 'rates of change' in the intensities of the image. These will be high where there are 'edges' in the image. An alternative method of edge detection is the Binary Edge Detector, a network for which can be seen on the ShrinkWhite help page.
