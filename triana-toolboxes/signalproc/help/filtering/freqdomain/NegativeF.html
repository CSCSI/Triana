Description of NegativeF

The unit called NegativeF takes an input spectral data set and returns the same data set with all the positive-frequency elements (including zero-frequency) set to zero. If the input set is one-sided, then it is converted to two-sided before being processed. See FullSpectrum for the effect of converting a one-sided spectrum to two-sided.
 

Using NegativeF

NegativeF has no parameter window.
