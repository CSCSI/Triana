Description of Name

The unit called Name accepts any GraphType data type as input and outputs a Parameter type that contains the title character string contained in the input data. This string is normally used as the title on a graph of the data. The output Parameter can be used as input to another unit, such as SetName.

Using Name

Name has no parameter window.
