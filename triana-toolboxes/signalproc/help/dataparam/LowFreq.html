Description of LowFreq

The unit called LowFreq accepts any data type that implements the Spectral interface as input and outputs a Parameter type that contains the lowest absolute value of the frequency contained in the input data. This can be input to another unit as a parameter.

Using LowFreq

LowFreq does not have a parameter window.
