Description of SetLabels

The unit called SetLabels accepts a SampleSet as input and outputs the same data, but with the axis labels of the graph set to the values given in the parameter window (see below). This will be used for example, by the Grapher as the axis labels for the graph when the data are displayed.

Using SetLabels

SetLabels's parameter window (double-click on the unit) is used to specify the axis labels to be entered into the data set. This resets the axis labels of the data to the given character string. A screen-shot of the parameter window with text entered into it is shown below:
