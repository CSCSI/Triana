Description of TestTwoSided

The unit called TestTwoSided accepts any data type that implements the Spectral interface as input and outputs a Parameter type that contains the a Boolean value of true if the input is two-sided and false if one-sided. This can be input to another unit as a parameter.
 

Using TestTwoSided

TestTwoSided has no parameter window.
