Description of SetName

The unit called SetName accepts any data type of type GraphType as input and outputs the same data, but with the title field (name of the data set) set to the value given in the parameter window (see below). This will be used for example, by the Grapher as the title of the graph when the data are displayed.

Using SetName

SetName's parameter window (double-click on the unit) is used to specify the name to be entered into the data set. This resets the title field of the data to the given character string.
