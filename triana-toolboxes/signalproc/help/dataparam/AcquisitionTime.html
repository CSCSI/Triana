Description of AcquisitionTime

The unit called AcquisitionTime accepts any data type that implements the Signal interface as input and outputs a Parameter type that contains the acquisition time of the input data. This can be input to another unit as a parameter, such as SetAcqTime. If the input is not of Signal type, then a text parameter is output with an error message.
 

Using AcquisitionTime

AcquisitionTime has no parameter window.
