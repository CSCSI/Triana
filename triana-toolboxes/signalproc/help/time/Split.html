Description of Split

The unit called Split takes in a SampleSet input and splits it into the number of segments specified using the user interface. For example, setting this value to 5 would split your time-series into 5 chunks. It will receive one chunk and then output the smaller chunks one by one until the complete series has been output. It then loads the next and so on.
 

Using Split

Split's parameter window (double-click on the unit while holding down the Control key) is used to set the number of chunks you wish to split your sample set into.
