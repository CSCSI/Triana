Description of SGTGrapher

The unit called SGTGrapher is a 2 dimensional plotting tool based in Java. It allows for the plotting of several line and scatter plots on the same area. It also allows for the cosmetics of the graph to be changed such as the colours used and the title names.
 

Using SGTGrapher

SGTGraphers's parameter window (double-click on the unit while holding down the Control key) is used to view the resultant graphs as well as make changes to them. Once a taskgraph has been run SGT will display the produced graph in a window that appears from within the parameter panel (see figur). From within this panel the graph can also be changed. Firstly we can use this panel to zoom in on certan aras of the graph that may be of interest. To do this use the mouse to drag a selection area arund the area which you wish to zoom in on and release. To reset the zoom select the Reset Zoom button.

Under the file menu is the choice to print the graph. The graph can be printed to a file or printer and works similarly to print functins available in other software.

The plot menu contains the methods to adjust the plot itself. Firstly it gives the option of amending the plot properties. In selecting this a window appears containing the information concerning the plot. Such information lncludes details on the axis and the titles. These can all be amended using this window, in a variety of ways. For example, the window can be used to change the mark type, colour, line type and tick type of each set of data in the plot.

The Line Key Title option under the plot menu allows the user to amend the line key value. This function is also available in the plot menu. The menu only works for the first line key.

Finally under the plot mennu there is the option to chnage the x or y axis to represent linear or log values.
