Description of GraceGrapher

The unit called GraceGrapher is a wrapping tool for the open soruce software Grace. Grace is a WYSIWYG two dimensional plotting tool for the x window System. It can be used to develop 2 dimensional coulor coded graphs. This unit arranges the data into a format that can be read into Grace.
 

Using GraceGrapher

Ensure that x11 is running on the machine and that Grace is installed, before attempting to use this unit. when the task graph is run the unit will attempt to run Grace, passing to it the data being used. Once this is done Grace will appear with the required plot (see figure). From this point onwards you are directed to the Grace website for further information concerning Graces functionality.
