Noise
Author : Ian Taylor
Type : ProcessingUnit
Input Types : SampleSet
Output Types : SampleSet
Date : 25 April 1996

The Noise is a unit which adds random noise to an input SampleSet. The amount of noise can be adjusted using the Noise's unit window. By default its value is one i.e. the level of noise is the same amplitude as the signal. This noise level can be changed by using the scrollbar at the top of the window or by editing the text field just below it. The minimum and maximum range for the scrollbar can be set using the two text fields at the bottom of the window.
