Description of OneSide

The unit called OneSide takes a data type containing a spectrum as input and outputs a one-sided spectrum. The returned data are the positive-frequency part of the spectrum.

This is useful especially for preparing spectral data for graphing, where the negative frequencies are often redundant. It also reduces the size of the stored data set. If the input data set is a Spectrum or ComplexSpectrum that was derived from the Fourier transform of a real data set, then no information is lost on conversion to a one-sided set. The unit called FullSpectrum inverts this and restores the original data set.
 

Using OneSide

OneSide has no parameter window.
