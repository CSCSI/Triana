The SpecAver Unit

The SpecAver is a unit which averages a specified number of spectra. The number of spectra to average can be changed by using the slider at the top of this window or by editing the text field just below it. The minimum and maximum limits on the number of spectra to average can be edited by changed the values which appear in the two text fields shown at the bottom of the window.

You would want to use a spectral averager in situations when you are searching for a constant signal within a noisy background. As you increase the number of spectra which are being averaged the more the noise effectively cancels itself out (since noise is random) and hence the clearer the signal becomes.
